##Object Repository File Rules Doc

AlphaStreet's Object Repository XML following Rules:-

		Images:			<identicalName>_Img
		Boxes:			<identicalName>_TBox,<identicalName>_RBtn,<identicalName>_CBox
		Buttons:		<identicalName>_Btn
		labels:			<identicalName>_Lbl
		Text:			<identicalName>_Txt
		DropDown:		<identicalName>_DD
			-List Item 		<identicalName>_DD_LI
		DIV's:			<identicalName>_Div
		Links's:		<identicalName>_Lnk
